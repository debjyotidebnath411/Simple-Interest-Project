# Simple Interest Project

A simple shell script to compute **Simple Interest**.

## Formula
Simple Interest = (P × R × T) / 100  
- **P** = Principal (amount)  
- **R** = Annual interest rate (percent)  
- **T** = Time (in years)

## Example
```bash
./simple-interest.sh 1000 5 2
# Simple Interest: 100.00
# Amount: 1100.00
